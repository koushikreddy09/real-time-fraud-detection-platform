# Tradeoffs

## Why Kafka?
Kafka allows asynchronous communication between services and fits high-throughput event pipelines.

## Why Redis?
Redis provides fast access to recent transaction counters and hot fraud features.

## Why PostgreSQL?
PostgreSQL provides durable storage for transactions, decisions, and alerts.

## Why Docker Compose first?
Docker Compose is the fastest and cheapest way to build and test the system locally before cloud deployment.
